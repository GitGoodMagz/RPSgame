Data Privacy Policy

Student assignment.

The application collects only the data needed to run the service.

This includes username, password hash, account creation time, and whether and when the Terms of Service were accepted.

Passwords are not stored in plain text. They are stored as secure hashes.

Data is stored in the application database and used only for application functionality. It is not sold or used for commercial purposes.

Users can view available account information in the application and can delete their account at any time. Deleting an account removes the related user data from the application database.

An account can only be created after the user accepts the Terms of Service. Consent can be withdrawn by deleting the account.
